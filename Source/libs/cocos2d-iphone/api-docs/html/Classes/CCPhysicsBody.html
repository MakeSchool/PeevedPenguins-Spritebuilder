surfaceVelocity

Velocity of the surface of the object relative to it’s normal velocity. This is useful for modeling conveyor belts or the feet of a player character. The calculated surface velocity of two colliding shapes by default only affects their friction. The calculated value can be overriden in a CCCollisionPairDelegate pre-solve method. Defaults to CGPointZero.
